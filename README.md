# Qt程序打包工具
> 基于Windows系统的Qt打包程序

## 适用范围
* Window系统;
* Qt5.0版本以上编译的程序。

## 使用方法
1. 将需要打包的程序拖拽到打包工具中；
2. 选择该程序编译时的Qt版本和编译器版本；
3. 点击生成；
4. 最后测试。
